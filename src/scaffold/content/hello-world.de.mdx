Willkommen zu deinem brandneuen Blog! Dies ist ein Beispielbeitrag, der von [CitePo](https://citepo.com) generiert wurde, um dir den Einstieg zu erleichtern.

## Was ist CitePo?

CitePo ist eine leichtgewichtige Blog-Publishing-Plattform. Du schreibst Beiträge in **MDX** (Markdown + JSX), pushst zu Git, und deine Seite wird automatisch gebaut und deployed.

<Callout type="tip">
  MDX ermöglicht dir, reichhaltige Komponenten in Markdown zu verwenden. Dieser Beitrag selbst ist eine MDX-Datei — scrolle zum Abschnitt "MDX-Komponenten", um zu sehen, was verfügbar ist!
</Callout>

## Deinen Ersten Echten Beitrag Schreiben

Erstelle eine neue `.mdx`-Datei im `content/`-Verzeichnis. Jeder Beitrag benötigt **Frontmatter** am Anfang:

```mdx
---
title: Mein Toller Beitrag
description: Eine kurze Zusammenfassung des Beitrags
date: 2025-01-27
tags: [tech, tutorial]
---

Dein Markdown-Inhalt kommt hier...
```

### Unterstützte Frontmatter-Felder

| Feld          | Erforderlich | Beschreibung                              |
| ------------- | ------------ | ----------------------------------------- |
| `title`       | Ja           | Der Titel des Beitrags                    |
| `description` | Ja           | Eine kurze Zusammenfassung (für SEO)      |
| `date`        | Ja           | Veröffentlichungsdatum (YYYY-MM-DD)       |
| `tags`        | Nein         | Array von Tags zur Kategorisierung        |
| `authors`     | Nein         | Array von Autorennamen                    |
| `coverImage`  | Nein         | Pfad zum Titelbild in `asset/`            |
| `draft`       | Nein         | Setze `true` um in Produktion zu verbergen|

## Formatierungsbeispiele

### Textformatierung

Du kannst **fett**, _kursiv_, ~~durchgestrichen~~ und `Inline-Code` schreiben.

### Code-Blöcke

```typescript
function grüßen(name: string): string {
  return `Hallo, ${name}! Willkommen bei CitePo.`
}

console.log(grüßen('Welt'))
```

### Listen

- Schreibe in deinem Lieblingseditor (VS Code, Cursor, Vim...)
- Vorschau mit `npx citepo dev`
- Baue mit `npx citepo build`
- Deploye durch Push zu Git
