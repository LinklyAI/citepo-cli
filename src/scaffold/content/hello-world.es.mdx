¡Bienvenido a tu nuevo blog! Esta es una publicación de ejemplo generada por [CitePo](https://citepo.com) para ayudarte a comenzar.

## ¿Qué es CitePo?

CitePo es una plataforma ligera de publicación de blogs. Escribes publicaciones en **MDX** (Markdown + JSX), haces push a Git, y tu sitio se construye y despliega automáticamente.

<Callout type="tip">
  MDX te permite usar componentes enriquecidos dentro de Markdown. ¡Esta publicación es un archivo MDX — desplázate hasta la sección "Componentes MDX" para ver lo que está disponible!
</Callout>

## Escribiendo Tu Primera Publicación Real

Crea un nuevo archivo `.mdx` en el directorio `content/`. Cada publicación necesita **frontmatter** al inicio:

```mdx
---
title: Mi Publicación Increíble
description: Un breve resumen de la publicación
date: 2025-01-27
tags: [tech, tutorial]
---

Tu contenido markdown va aquí...
```

### Campos de Frontmatter Soportados

| Campo         | Requerido | Descripción                              |
| ------------- | --------- | ---------------------------------------- |
| `title`       | Sí        | El título de la publicación              |
| `description` | Sí        | Un breve resumen (usado en SEO)          |
| `date`        | Sí        | Fecha de publicación (YYYY-MM-DD)        |
| `tags`        | No        | Array de etiquetas para categorización   |
| `authors`     | No        | Array de nombres de autores              |
| `coverImage`  | No        | Ruta a imagen de portada en `asset/`     |
| `draft`       | No        | Establece `true` para ocultar en producción |
