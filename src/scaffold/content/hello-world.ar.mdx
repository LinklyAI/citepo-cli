مرحباً بك في مدونتك الجديدة! هذه مقالة نموذجية أنشأها [CitePo](https://citepo.com) لمساعدتك على البدء.

## ما هو CitePo؟

CitePo هو منصة خفيفة لنشر المدونات. تكتب المقالات بـ **MDX** (Markdown + JSX)، تدفع إلى Git، ويتم بناء ونشر موقعك تلقائياً.

<Callout type="tip">
  MDX يتيح لك استخدام مكونات غنية داخل Markdown. هذه المقالة نفسها ملف MDX — انتقل إلى قسم "مكونات MDX" لترى ما هو متاح!
</Callout>

## كتابة أول مقالة حقيقية

أنشئ ملف `.mdx` جديد في مجلد `content/`. كل مقالة تحتاج **frontmatter** في الأعلى:

```mdx
---
title: مقالتي الرائعة
description: ملخص موجز للمقالة
date: 2025-01-27
tags: [تقنية, درس]
---

محتوى markdown الخاص بك هنا...
```

### حقول Frontmatter المدعومة

| الحقل         | مطلوب | الوصف                              |
| ------------- | ----- | ---------------------------------- |
| `title`       | نعم   | عنوان المقالة                      |
| `description` | نعم   | ملخص قصير (يستخدم للـ SEO)         |
| `date`        | نعم   | تاريخ النشر (YYYY-MM-DD)           |
| `tags`        | لا    | مصفوفة وسوم للتصنيف                |
| `authors`     | لا    | مصفوفة أسماء الكتاب                |
| `coverImage`  | لا    | مسار صورة الغلاف في `asset/`       |
| `draft`       | لا    | اضبط `true` للإخفاء في الإنتاج     |

## أمثلة التنسيق

### تنسيق النص

يمكنك كتابة **عريض**، _مائل_، ~~مشطوب~~، و`كود مضمن`.

### كتل الكود

```typescript
function greet(name: string): string {
  return `مرحباً، ${name}! أهلاً بك في CitePo.`
}

console.log(greet('العالم'))
```

### القوائم

- اكتب في محررك المفضل (VS Code، Cursor، Vim...)
- معاينة بـ `npx citepo dev`
- بناء بـ `npx citepo build`
- نشر بالدفع إلى Git

### اقتباسات

> أفضل وقت لبدء مدونة كان أمس.
> ثاني أفضل وقت هو الآن.
