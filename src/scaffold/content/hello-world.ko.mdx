새 블로그에 오신 것을 환영합니다! 이것은 [CitePo](https://citepo.com)가 시작을 돕기 위해 생성한 샘플 게시물입니다.

## CitePo란?

CitePo는 가벼운 블로그 퍼블리싱 플랫폼입니다. **MDX**(Markdown + JSX)로 게시물을 작성하고, Git에 푸시하면 사이트가 자동으로 빌드되고 배포됩니다.

<Callout type="tip">
  MDX를 사용하면 Markdown 안에서 풍부한 컴포넌트를 사용할 수 있습니다. 이 게시물 자체가 MDX 파일입니다 — "MDX 컴포넌트" 섹션으로 스크롤하여 사용 가능한 것들을 확인하세요!
</Callout>

## 첫 번째 실제 게시물 작성하기

`content/` 디렉토리에 새 `.mdx` 파일을 만드세요. 모든 게시물은 상단에 **frontmatter**가 필요합니다:

```mdx
---
title: 멋진 게시물
description: 게시물의 간단한 요약
date: 2025-01-27
tags: [tech, tutorial]
---

마크다운 내용이 여기에 들어갑니다...
```

### 지원되는 Frontmatter 필드

| 필드          | 필수 | 설명                               |
| ------------- | ---- | ---------------------------------- |
| `title`       | 예   | 게시물 제목                        |
| `description` | 예   | 짧은 요약 (SEO에 사용)             |
| `date`        | 예   | 게시 날짜 (YYYY-MM-DD)             |
| `tags`        | 아니오 | 분류를 위한 태그 배열              |
| `authors`     | 아니오 | 작성자 이름 배열                   |
| `coverImage`  | 아니오 | `asset/`의 커버 이미지 경로        |
| `draft`       | 아니오 | 프로덕션에서 숨기려면 `true` 설정  |
