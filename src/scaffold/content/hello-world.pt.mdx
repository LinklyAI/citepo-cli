Bem-vindo ao seu novo blog! Este é um post de exemplo gerado pelo [CitePo](https://citepo.com) para ajudá-lo a começar.

## O que é CitePo?

CitePo é uma plataforma leve de publicação de blogs. Você escreve posts em **MDX** (Markdown + JSX), faz push para o Git, e seu site é construído e implantado automaticamente.

<Callout type="tip">
  MDX permite usar componentes ricos dentro do Markdown. Este post é um arquivo MDX — role até a seção "Componentes MDX" para ver o que está disponível!
</Callout>

## Escrevendo Seu Primeiro Post Real

Crie um novo arquivo `.mdx` no diretório `content/`. Todo post precisa de **frontmatter** no topo:

```mdx
---
title: Meu Post Incrível
description: Um breve resumo do post
date: 2025-01-27
tags: [tech, tutorial]
---

Seu conteúdo markdown vai aqui...
```

### Campos de Frontmatter Suportados

| Campo         | Obrigatório | Descrição                                |
| ------------- | ----------- | ---------------------------------------- |
| `title`       | Sim         | O título do post                         |
| `description` | Sim         | Um breve resumo (usado em SEO)           |
| `date`        | Sim         | Data de publicação (YYYY-MM-DD)          |
| `tags`        | Não         | Array de tags para categorização         |
| `authors`     | Não         | Array de nomes de autores                |
| `coverImage`  | Não         | Caminho para imagem de capa em `asset/`  |
| `draft`       | Não         | Defina `true` para ocultar em produção   |

